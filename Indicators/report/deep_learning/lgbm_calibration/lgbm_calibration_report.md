# LightGBM 购买概率模型：概率校准报告

## 1. 报告背景

上一阶段已经完成了 LightGBM 购买概率模型实验，并选出了最优版本。LightGBM 相比逻辑回归在排序能力和正样本识别能力上有明显提升，但树模型输出的概率值不一定等于真实业务购买概率。

因此，本阶段继续对 LightGBM 最优模型做概率校准。目标是保留 LightGBM 的排序能力，同时让输出概率更接近真实购买率。

本次输入文件为：

```text
lgbm_predictions_best_with_path.csv
```

其中包含 valid 和 test 数据集的真实标签 `label_buy_7d` 以及 LightGBM 原始预测概率 `lgbm_probability`。

## 2. 校准方法

本次使用 valid 集拟合校准器，然后在 valid 和 test 上评估校准效果。使用了两种方法：

| 方法     | 说明                  | 适用情况                                 |
| -------- | --------------------- | ---------------------------------------- |
| original | LightGBM 原始输出概率 | 适合排序，但概率可能偏高或偏低           |
| platt    | Platt Scaling         | 用逻辑回归校准概率，稳定性较好           |
| isotonic | Isotonic Regression   | 非参数单调校准，拟合能力强，但可能过拟合 |

本次校准的核心逻辑是：

```text
LightGBM 负责学习排序
校准器负责修正概率数值
```

## 3. 整体校准指标

本次重点看 AUC、PR-AUC、Brier Score、Log Loss 和 ECE。其中 AUC、PR-AUC 主要反映排序能力，Brier Score、Log Loss 和 ECE 主要反映概率是否可信。Brier Score、Log Loss、ECE 越低，说明概率越接近真实购买率。

| 数据集 | 方法     |    AUC | PR-AUC | Brier Score | Log Loss |    ECE | 平均预测概率 | 真实购买率 |
| ------ | -------- | -----: | -----: | ----------: | -------: | -----: | -----------: | ---------: |
| valid  | original | 0.9056 | 0.5673 |      0.1106 |   0.3569 | 0.1826 |       27.40% |      9.15% |
| valid  | platt    | 0.9056 | 0.5673 |      0.0547 |   0.1894 | 0.0018 |        9.14% |      9.15% |
| valid  | isotonic | 0.9068 | 0.5586 |      0.0543 |   0.1878 | 0.0000 |        9.15% |      9.15% |
| test   | original | 0.9232 | 0.6348 |      0.1101 |   0.3547 | 0.1884 |       27.88% |      9.04% |
| test   | platt    | 0.9232 | 0.6348 |      0.0490 |   0.1718 | 0.0104 |        9.80% |      9.04% |
| test   | isotonic | 0.9228 | 0.6197 |      0.0492 |   0.1749 | 0.0110 |        9.82% |      9.04% |

从 test 集结果看，LightGBM 原始平均预测概率为 27.88%，真实购买率为 9.04%。Platt 校准后平均预测概率为 9.80%，Isotonic 校准后平均预测概率为 9.82%。

这说明校准后概率整体更接近真实购买率，尤其适合后续做业务解释和转化率预估。

## 6. 结果解读

本阶段可以得到两个结论。

第一，LightGBM 原始概率更适合作为排序分数。它可以帮助我们找出最可能购买的用户-商品组合，因此适合推荐排序、营销触达候选池和 Top 用户商品筛选。

第二，校准后的概率更适合作为真实购买概率解释。如果业务场景需要回答“这个用户购买这个商品的概率大约是多少”，就不应该直接使用原始 `lgbm_probability`，而应该使用校准后的概率。

综合 test 集 Brier Score、Log Loss 和 ECE，本次表现最好的校准方式是 `platt`。

## 7. 总结

本阶段完成了 LightGBM 最优模型的概率校准。结果表明，LightGBM 原始输出更适合用于排序，校准后的概率更适合用于业务解释和转化率预估。后续建议保留原始概率用于推荐排序，保留 Platt 校准概率用于购买概率解释，并进入最终模型汇总和项目报告阶段。
