# XGBoost 购买概率模型：概率校准报告

## 1. 报告背景

前面已经完成了 XGBoost 购买概率模型实验，并得到最优 XGBoost 版本。XGBoost 的排序能力较强，但树模型输出的原始概率不一定等于真实业务购买概率。因此，本阶段对 XGBoost 最优模型进行概率校准。

本次输入文件为：

```text
xgb_predictions_best_with_path.csv
```

该文件包含 valid 和 test 两个数据集的真实标签 `label_buy_7d`，以及 XGBoost 原始预测概率 `xgb_probability`。

本阶段目标是：

```text
保留 XGBoost 的排序能力
同时让输出概率更接近真实购买率
```

## 2. 校准方法

本次使用 valid 集拟合校准器，然后在 valid 和 test 集上评估校准效果。使用了两种校准方法：

| 方法         | 说明                 | 适用情况                                 |
| ------------ | -------------------- | ---------------------------------------- |
| `original` | XGBoost 原始输出概率 | 适合排序，但概率可能偏高                 |
| `platt`    | Platt Scaling        | 用逻辑回归校准概率，稳定性较好           |
| `isotonic` | Isotonic Regression  | 非参数单调校准，拟合能力强，但可能过拟合 |

## 3. 整体校准指标

本次重点看 AUC、PR-AUC、Brier Score、Log Loss 和 ECE。AUC、PR-AUC 主要看排序能力；Brier Score、Log Loss 和 ECE 主要看概率是否可信。后三个指标越低，说明概率越接近真实购买率。

| 数据集 | 方法         |    AUC | PR-AUC | Brier Score | Log Loss |    ECE | 平均预测概率 | 真实购买率 |
| ------ | ------------ | -----: | -----: | ----------: | -------: | -----: | -----------: | ---------: |
| valid  | `original` | 0.9037 | 0.5621 |      0.1114 |   0.3630 | 0.1890 |       28.05% |      9.15% |
| valid  | `platt`    | 0.9037 | 0.5621 |      0.0550 |   0.1908 | 0.0026 |        9.15% |      9.15% |
| valid  | `isotonic` | 0.9049 | 0.5537 |      0.0546 |   0.1891 | 0.0000 |        9.15% |      9.15% |
| test   | `original` | 0.9221 | 0.6380 |      0.1108 |   0.3610 | 0.1953 |       28.57% |      9.04% |
| test   | `platt`    | 0.9221 | 0.6380 |      0.0490 |   0.1721 | 0.0100 |        9.88% |      9.04% |
| test   | `isotonic` | 0.9218 | 0.6212 |      0.0492 |   0.1741 | 0.0114 |        9.84% |      9.04% |

从 test 集结果看，XGBoost 原始平均预测概率为 28.57%，真实购买率为 9.04%。Platt 校准后平均预测概率为 9.88%，Isotonic 校准后平均预测概率为 9.84%。

这说明 XGBoost 原始概率存在偏高问题，经过校准后，平均预测概率已经明显更接近真实购买率。

## 6. 结果解读

本阶段可以得到两个结论：

```text
1. XGBoost 原始概率更适合做排序分数。
2. 校准后的概率更适合做真实购买概率解释。
```

综合 test 集 Brier Score、Log Loss 和 ECE，本次表现最好的校准方式是 `platt`。不过从稳定性角度看，Platt 通常更稳，因此如果要在业务汇报中解释购买概率，建议优先使用 `platt_xgb_probability`。

## 7. 推荐使用方式

| 使用场景                   | 推荐字段                     | 原因                             |
| -------------------------- | ---------------------------- | -------------------------------- |
| 推荐排序、Top 用户商品筛选 | `original_xgb_probability` | 排序能力稳定                     |
| 业务购买概率解释           | `platt_xgb_probability`    | 概率更接近真实购买率，稳定性较好 |
| 校准对照分析               | `isotonic_xgb_probability` | 可作为辅助参考                   |
